¿Quién soy?

Gracias por visitar mi sitio web personal, mi nombre es Germán Brucaglia, tengo 29 años de edad y soy de Rosario, Santa Fe - Argentina. Soy egresado de la carrera de Analista de Sistemas de Computación y me oriento hacia el desarrollo web. Aunque mi fuerte es el desarrollo front end tambien poseo conocimientos back end y creo que estoy a la altura para desempeñarme de manera eficaz en cualquier puesto. Me considero una persona pro activa, con exelente relación con los demas capaz de rendir en un alto nivel en cualquier tarea que se me asigne. Lo que mas me gusta es diseñar y crear sitios web dinamicos, responsivos y que le permitan al usuario tener una experiencia agradable al navegar por ellos.

Estudios y cursos

Titulo en Analista de Sistemas de Computación, durante toda la carrera aprendi sobre hardware, redes, conexiones, sistemas operativos, desarrollo front end y back end entre otras cosas.

Actualmente me encuentro realizando el curso de desarrollo web utilizando HTML, CSS, JavaScript, bootstrap, NodeJS, Git y GitHub.

Finalicé este Labs para la empresa Neoris, en el se vieron tecnologías como C#, .Net Framework, MVC, Web Api, Sql, Entity Framework, Angular, HTML, CSS, JavaScript, Bootstrap entre otros.

Finalicé el curso de Introducción a la programación web al ganarme una beca otorgada por la empresa Neoris. En el mismo se debió entregar un proyecto utilizando HTML, CSS, JavaScript y Bootstrap

Realicé el curso de Desarrollo web con JavaScript, en el que se vieron temas como conceptos básicos de JS, manejo del DOM, POO, Ajax Y Websockets entre otros.
